# Skill: Test Generator
## Goal
Create comprehensive test suites.

## Procedures
1. **Coverage:** Target >90% code coverage.
2. **Types:** Mix of unit, integration, and E2E tests.
3. **Edge Cases:** Explicit testing of error conditions and boundary values.
4. **Mocks:** Effective use of mocks/stubs for external dependencies.
5. **Fixtures:** Maintain clean and reusable test fixtures.
