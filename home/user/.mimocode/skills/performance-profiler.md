# Skill: Performance Profiler
## Goal
Identify and resolve system performance bottlenecks.

## Procedures
1. **Bottlenecks:** CPU-bound tasks, memory leaks, blocking I/O.
2. **Metrics:** Time complexity analysis.
3. **Tools:** Chrome DevTools, Node inspector, flame graphs.
4. **Action:** Suggest code-level optimizations (algorithms, memoization, lazy loading).
5. **Validation:** Compare before/after metrics.
