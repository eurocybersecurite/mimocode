# Skill: CI/CD Pipeline
## Goal
Generate automated deployment pipelines.

## Procedures
1. **Scope:** Define steps (Lint, Test, Build, Security Scan, Deploy).
2. **Strategy:** Support Blue/Green or Canary deployments if possible.
3. **Environment:** Secrets management (Env variables, secrets).
4. **Integration:** Notification hooks (Slack/Email).
5. **Validation:** Ensure the pipeline is idempotent and failures stop the process.
