# Skill: Database Optimizer
## Goal
Improve database performance and schema reliability.

## Procedures
1. **Analysis:** Identify missing indexes and N+1 query problems.
2. **Schema:** Normalize schema and validate data types.
3. **Execution:** Optimize queries (EXPLAIN ANALYZE).
4. **Caching:** Suggest Redis/memcached strategies for heavy read workloads.
5. **Security:** Prevent SQL injection and ensure least-privilege access.
