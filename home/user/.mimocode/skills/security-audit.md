# Skill: Security Audit
## Goal
Ensure code security and vulnerability remediation.

## Procedures
1. **Scanning:** Check for SQL Injection, XSS, CSRF.
2. **Secrets:** Scan for hardcoded credentials or API keys.
3. **Authentication:** Validate auth flow and token security.
4. **Traversal:** Check path traversal and system command injection.
5. **Report:** Provide CVE-style severity ratings and patches.
