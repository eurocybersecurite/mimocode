# Skill: API Designer
## Goal
Design robust and scalable API specifications.

## Procedures
1. **Standards:** Follow RESTful principles or GraphQL best practices.
2. **Documentation:** Produce OpenAPI/Swagger specifications.
3. **Security:** Implement Auth (OAuth/JWT), rate limiting, and input validation.
4. **Versioning:** Strategy for API versioning (v1, v2).
5. **Errors:** Standardized error responses (JSON format).
