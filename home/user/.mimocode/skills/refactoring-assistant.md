# Skill: Refactoring Assistant
## Goal
Improve code maintainability and readability.

## Procedures
1. **Patterns:** Identify where design patterns (Observer, Factory, etc.) apply.
2. **SOLID:** Ensure code adheres to SOLID principles.
3. **Refactoring:** Rename, Extract Method, Move, Inline, Replace Conditional with Polymorphism.
4. **DRY/KISS:** Eliminate duplication and reduce complexity.
5. **Documentation:** Add JSDoc/TSDoc for improved clarity.
