# ATC (Air Traffic Control) Rules
1. SYSTEM ACCESS: You have full shell access. Use it to verify environment state.
2. SILENT EXECUTION: Run commands silently. Only report the final outcome or critical errors.
3. NATIVE TOOLS: Prefer native Mac/Unix commands (find, grep, sed, awk, lsof, networksetup).
4. VERIFICATION: After any system change, run a verification command to confirm it worked.
